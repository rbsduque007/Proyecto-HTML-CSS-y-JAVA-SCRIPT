CSS

¿Qué es CSS?

CSS es un lenguaje de diseño que permite dar estilo y formato a las páginas web.

¿Para qué sirve?

Sirve para cambiar colores, tamaños, posiciones, fondos, bordes, animaciones y más.

ESTRUCTURA BASICA DE CSS

La estructura fundamental de CSS se compone de tres partes principales: el selector, las propiedades y los valores. Esta forma de escribir reglas permite definir cómo deben mostrarse los elementos dentro de una página web. Una regla de CSS sigue el siguiente formato:

selector {

propiedad: valor;

}

El selector indica qué elemento HTML será modificado, como un párrafo, un encabezado o una clase específica. Las propiedades representan las características que se desean cambiar, como el color del texto, el tamaño de letra, el margen o el ancho.

Cambios históricos

CSS ha evolucionado mucho desde su creación para mejorar la apariencia y el diseño de las páginas web. Su historia se divide en varias etapas importantes:

  • 1. CSS1 (1996) : La primera versión. Permitía cambiar colores, fuentes, márgenes y alineación básica. Era sencillo y limitado, pero sentó las bases del diseño web.
  • 2. CSS2 (1998) : Añadió funciones más avanzadas como posicionamiento absoluto y relativo, estilos para distintos medios (pantalla, impresión) y mejor control de las capas. Permitió diseños más complejos.
  • 3. CSS2.1 (2004–2011) : Corrige errores y define la versión estable del estándar. Mejoró la compatibilidad entre navegadores, lo cual fue muy importante.
  • 4. CSS3 (2011–actualidad) :La versión más grande y la que usamos hoy. Se divide en módulos independientes (colores, fondos, animaciones, diseño flexible, variables, etc.).

    • Incluye grandes avances como :

  • Transiciones y animaciones
  • Sombras y bordes redondeados
  • Flexbox y Grid para mejores diseños
  • Transformaciones 2D y 3D
  • Nuevos tipos de colores y efectos

5. CSS4 (concepto en desarrollo) :

No existe una versión “CSS4” oficial, pero sí módulos nuevos y más avanzados, como selectores más potentes y funciones modernas. La idea es que CSS siga creciendo por módulos, no por versiones completas.
