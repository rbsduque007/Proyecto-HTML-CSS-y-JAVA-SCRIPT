¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar interacción y dinamismo a las páginas web.

¿Para qué sirve?

Sirve para crear funciones, animaciones, validaciones, efectos visuales, controlar eventos y manipular el contenido de una página en tiempo real.

ESTRUCTURA BÁSICA DE JAVASCRIPT

La estructura fundamental de JavaScript se compone de instrucciones que el navegador interpreta para realizar acciones. Una instrucción básica puede declarar variables, ejecutar operaciones o llamar funciones.

Ejemplo básico:

let mensaje = "Hola";

alert(mensaje);

Las variables almacenan datos, las funciones agrupan instrucciones y los eventos permiten reaccionar a acciones del usuario como clics o teclas.

Cambios históricos

JavaScript ha evolucionado desde 1995 para convertirse en uno de los lenguajes más importantes del mundo.

1. JavaScript (1995)

Creado por Brendan Eich en solo 10 días. Su objetivo inicial era agregar interactividad básica.

2. Estándar ECMAScript (1997)

Se convirtió en un lenguaje estandarizado para asegurar compatibilidad entre navegadores.

3. ES5 (2009)

Incluyó mejoras como JSON, modo estricto y funciones más seguras.

4. ES6 / ES2015 (2015)

La actualización más grande: let, const, clases, promesas, funciones flecha, módulos y más.

5. Actualizaciones anuales (2016–actualidad)

JavaScript ahora mejora cada año con nuevas funciones modernas como async/await, operadores avanzados y APIs del navegador.
